# Fullnode Base Operator

Start and scale out cypto Blockchain fullnode easily

# Multi-chain support
support BSC / ETH / SOL / AVAX / Ploygon

# Examples

```
# BSC-node.yaml
apiVersion: bsc.blockchain.io/v1alpha1
kind: Node
metadata:
  name: bsc-node
spec:
  network: mainnet
  role: p2p
```
